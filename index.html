The Mission Pinball Framework (“MPF”) is a free Python-based pinball software framework that’s used to run real pinball machines. It allows both casual builders and hard-core programmers to create the software to run their pinball machines—whether it’s new game code for an existing pinball machine, a “re-theme” of an old machine, or totally custom pinball or any homebrew pinball machine built from scratch.

MPF is cross-platform and runs on Windows, Mac, Linux, and the Raspberry Pi. It's available in 32-bit and 64-bit versions and can be installed in minutes.


Build your game with config files or code

95%+ of what you need to do to program a pinball machine in MPF can be done with text-based configuration files, so even if you have no prior programming experience, you can still do it!

Our step-by-step tutorial walks you through the whole process, from listing your switches, coils, and lights, to building modes and scoring, and through creating graphics and sounds.

If you're a programmer and you know Python, we have an API reference, and it's really easy to mix-and-match custom code with the MPF config files. MPF provides two levels of abtraction: hardware abstraction and device abstraction which both can be used in code. Configs work on top of those abstractions and add logic blocks and config players.

Hardware Independent

MPF supports many different pinball hardware controllers, including hardware from:

  • FAST Pinball
  • Open Pinball Project
  • Multimorphic P-ROC / P3-ROC
  • LISY 1 and LISY 80
  • Stern Spike
  • Your custom pinball platform. Use our interface to implement your protocol.

It also supports all sorts of other hardware, like the FadeCandy LED controllers, SmartMatrix RGB DMDs, Pololu Servo Controllers, Trinamics stepper controller and more.

MPF is written in a hardware-independent way, so we can easily add support for new hardware as it comes out. You can even swap between different kinds of hardware after your game is done!

Multiple eras of machines

You can use MPF with modern pinball control hardware to build your own custom "home brew" pinball machine from scratch.

The previously mentioned pinball hardware controllers can be swapped into existing machines (or in case of Stern SPIKE reprogrammed) in just a few minutes to completely remove the original code and control them via MPF, including:

  • Williams / Bally / Midway WPC
  • Williams System 11
  • Data East
  • Stern Whitestar, SAM, & SPIKE
  • Gottlieb System 1 and 80
  • Your custom pinball machine
You can also use MPF with any existing machine if you want to redo the wiring.

Pinball Mechs

MPF includes built-in support for just about every kind of pinball mechanism, including:

  • Switches
  • Coils
  • Lights
  • RGB LEDs
  • Flippers
  • Accelerometers
  • Troughs
  • Plungers & ball launch devices
  • VUKs, ball locks, ball capture devices
  • Targets (drop targets, standups, kicking targets)
  • GI
  • Flashers
  • Loops/orbits/ramps
  • Kickback lanes
  • Magnets
  • Motors
  • Multiple playfields
  • Pop bumpers
  • Mechanical score reels
  • Servos
  • Slingshots
  • Spinners
Game Logic

You can use MPF's config files to implement complex game features & logic, including:

  • Player management
  • Game modes & mode stacking
  • Ball locks
  • Ball saves
  • Ball search
  • Ball tracking
  • Extra balls
  • Multiballs
  • Timers
  • Shots & shot groups
  • Combo switches (flipper cancel, super skill shot, etc.)
  • End of ball bonus
  • Video modes
  • Player achievements
  • High scores
  • Tilt
  • Coins & credits
  • Synchronized shows of lights, sounds, & animations
Display, DMDs, & Graphics

MPF includes a powerful media controller based on Kivy to control graphics and sound. It leverages hardware GPU acceleration and OpenGL to provide a modern, smooth environment, with support for:

  • Mono 128x32 dot matrix displays (DMDs)
  • RGB LED "Smart Matrix" displays
  • LCDs & on-screen displays
  • Virtual "color" DMDs
  • Video
  • Images
  • Animations & motion

MPF also supports dynamically updating text, multi-language strings, and everything else you need to create a professional-looking machine.

Sounds & Audio

MPF includes a custom-built audio library built on SDL2 and GStreamer which was designed for pinball environments.

  • Multi-track audio
  • Track-based volume and sound control
  • Sound queue, sound expiration, and sound priorities
  • Brickwall limiting
  • Fade in / fade out
  • Instance limiting
  • Ducking
  • Attenuation
  • Sound variation
  • Event-driven live looping audio tracks with synchonized sound layers
  • Playlists
  • Stream sound files from disk, or load into memory
Completely free & open source

MPF is released under the “MIT” license which is extremely permissive. You can do just about anything you want with it. Use it. Copy it. Modify it. Merge, publish, distribute, sublicense and/or sell it. If you make a commercial pinball machine based on MPF, you do not have to pay us anything, and you do not have to release your machine’s source code if you don’t want to.

We don’t "own" MPF any more than you do. So go nuts. Rip us off. Take the code. Make it yours. Don’t give us credit… It’s all good! Seriously. We’re just putting the code out there which will hopefully lead to more pinball in the world!
